# Signalement de problemes de securite

Si vous pensez avoir identifie une **vulnerabilite de securite** liee aux installateurs ou au mecanisme de mise a jour publies dans ce depot, merci de **ne pas** ouvrir une issue publique avec les details.

## Piste recommandee

1. Verifiez si l'onglet **Security** du depot propose **Private vulnerability reporting** ; si oui, utilisez ce formulaire.
2. Sinon, contactez **Joseph Deffayet** par un **canal prive** (e-mail ou autre moyen qu'il vous aura communique).

Indiquez une description claire du probleme, les etapes de reproduction si possible, et la version concernee (numero visible dans le nom du fichier ou dans l'application).
